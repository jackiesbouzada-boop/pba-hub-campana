Los dos PDFs por municipio (Ministro + Subsecretario) permiten extraer automáticamente las siguientes métricas para construir el tablero provincial:

📄 Informe Ministro (4 págs) → 8 métricas clave
✓ Activos financieros en Bco. Provincia (último saldo)
✓ Relación Activos/Nómina Salarial (indicador de liquidez)
✓ Ingresos Corrientes, Propios y Transferencias (2025)
✓ Gastos Corrientes y de Capital (2025)
✓ Resultado Corriente y Financiero (2025)
✓ % Autonomía y % Gasto en Personal
✓ Deuda Flotante Neta de Salarios
✓ Planta de Personal y gasto medio por agente
📄 Informe Subsecretario (7 págs) → 12 métricas
✓ Serie histórica recursos/gastos 2016–2025
✓ Evolución deuda flotante 2016–2025
✓ CUD (Coeficiente Único de Distribución)
✓ Transferencias proyectadas 2026 por concepto
✓ Deuda con Provincia (Leasing) histórica
✓ Capacidad de ahorro, flexibilidad, inversión (ratios 2025)
✓ Apertura del gasto por función (Salud, Educación, Obras, etc.)
✓ Indicadores demográficos, productivos y sociales
✓ PBG y estructura económica local
✓ Precalificación endeudamientos
✓ Ayudas financieras recibidas (ATN, ATP, FEFIM)
✓ Paritaria municipal
